# PRODIGY_ML_05

## Food Recognition and Calorie Estimation

### Internship
Prodigy InfoTech Machine Learning Internship

### Objective
Develop a food recognition system that identifies food items from images and estimates their calorie content.

### Technologies Used
- Python
- Pillow (PIL)
- NumPy

### Features
- Detect food items from image names
- Estimate calories for each food item
- Process multiple images automatically

### Sample Foods
- Pizza
- Burger
- Apple

### Result
Successfully recognized food items and estimated their calorie values.
